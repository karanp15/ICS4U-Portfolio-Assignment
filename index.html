About Me!

My name is Alex Clements and I am a Grade 11 Student at A.Y. Jackson Secondary School in Ottawa, Ontario. In Grade 9, my first year of high school, I reached ahead and took the ICS2O course. In that class, I began to learn how to code and got familiar with many computer science concepts. I learned ActionScript and used it throughout that course. In Grade 10, I took the ICS3U course, this time learning the programming language C++. This semester, for Grade 11, I chose to take the ICS4U course. In this course, through working with both Khan Academy and freeCodeCamp, I have learned and developed my skills in many different languages, such as JavaScript, HTML, CSS, SQL, ES6 and Sass. As well, I worked with jQuery, Bootstrap, React, Redux, truly helping to show just how useful JavaScript libraries can be.

However, computer programming isn't my entire life (luckily). I am a huge hockey fan who loves the Boston Bruins (as you'll see in some of these projects). I also like the 3 other major North American sports (Go Packers, Celtics and Jays). On top of that, my favourite genre of music is EDM and my favourite musician of all time will forever be Avicii ◢ ◤.

My Top 10


The following 10 modules were my favourite and the most beneficial to me from the ICS4U course. These were created from modules developed by both Khan Academy and freeCodeCamp. I chose one project from each module that I felt was my best or the most interesting. You will clearly be able to see and interact with the projects down below. The paragraph that goes with each module just describes their benefits to me now as well as for the future.



Intro to HTML



Prior to this course, I had a decent understanding of both HTML and CSS and how they both work together to create a webpage. This Khan Academy course certainly helped to clean up my code and fix simple mistakes. As well, it taught me different ways of formatting webpages, which certainly helped in projects completed after the course, and, most importantly, in the making of the website you're on right now! The end of the course also gave me interesting tips on aspects of HTML often overlooked by beginner programmers and how to host and create your own website.




Intro to SQL



The Intro to SQL course from Khan Academy was quite valuable in teaching me how to create and use databases in computer programming. The course gave many useful tricks on how to best create a database and manage the data within it. It taught me how to store, query and change data within an SQL database. Through the four different projects in this course, I was able to design databases that stored many different sets of data. As well, I was then able to use and read the data within those databases. With the knowledge gained from working with SQL, I will be able to create and manage databases with efficiency in the future.




Intro to JS



While I did have quite a bit of experience in working with JavaScript from my own personal work, this Khan Academy course was quite beneficial in reinforcing and refreshing my understanding of how to best use JavaScript, even though there weren't many concepts in the course that I didn't understand before. As well, this course helped to truly show me the usefulness of adding in other libraries for JavaScript. In this case, Khan Academy used the ProcessingJS library to help visualize JavaScript through drawings and animations. It certainly bettered my ability to design visual aspects through code, and that will be useful for many years to come.




Natural Simulations



The Natural Simulations course from Khan Academy really helped to open my eyes and show me just how significant the relationship between math and computer science is. While I certainly can admit that I had a fair amount of trouble in working through this course, as it was quite complicated for me at times, it was more than beneficial to get through the entirety of it and learn so much more about computer science and mathematics. The most interesting aspect of this course was the Creature projects, that I had to keep building on at the end of some of the lessons. By the end of it, I had a colony of creatures that interacted with each other through different forces such as repulsion and gravity. For both mathematics and the sciences, the knowledge that I gained from this course will be extremely beneficial in the future.




Games & Visualizations



The Games & Visualizations course was perhaps the most interesting (and fun) module from Khan Academy for me. While I did already understand a majority of the content covered in the course, it gave me new ideas on how to code certain programs with JavaScript and brought a different enjoyment to coding for me. While there were only two projects for this course, they were both a lot of fun to complete and mess around with. While I don't know what my future holds, and whether being able to specifically design games would be useful, it did still teach me interesting aspects of visual design with JavaScript.




Making Webpages Interactive With jQuery



Perhaps Khan Academy's most important course when it comes to web design, this course really taught me how to save so much time when working with JavaScript. I learned that jQuery, which is a JavaScript library, thoroughly helps to allow for the use of the DOM (Document Object Model). I was able to learn even more ways to style webpages, as well as use animation to engage potential viewers. While I didn't fully enjoy coding in jQuery, I am certainly able to see and understand just how useful adding and using other JavaScript libraries is. That is the most important principle that I've taken from the Khan Academy JavaScript courses: JavaScript libraries are insanely beneficial add-ons that really help to get so many different things done better and more efficiently.




Responsive Web Design




Similarly to the Khan Academy HTML/CSS course, the Responsive Web Design course on freeCodeCamp added to my pre-existing knowledge of working with HTML and CSS. While I was familiar with both before working through this module and a few of its projects, it taught me how to best use HTML and CSS together and certainly helped to improve my artistic ability when it comes to web design. Following the completion of this module, I have been much better at designing pages and programs through code and have certainly benefited from it.




JavaScript Algorithms and Data Structures




This freeCodeCamp module helped to increase my understanding of JavaScript and furthered my abilities to successfully code programs in the language. While some of the concepts, such as Object-Oriented Programming and the Algorithm Scripting, were a lot tougher than others, I successfully completed those modules and learned a lot more about JavaScript. As well, I learned how to best use the different functions in javaScript in different scenarios. In addition, I was able to complete the certification on freeCodeCamp for this course, and really feel that I have a solid understanding of how to best code in JavaScript, which will be quite useful in the future.
Note: There were no available projects to display for this module, although I did get the complete certification.




Front End Libraries




This module for freeCodeCamp used a variety of different JavaScript libraries to further extend my knowledge of JavaScript and the languages related to it. For example, Bootstrap helped me to better write in CSS and JavaScript, allowing for more style options, just like Sass (although that is a language on its own). As well, the jQuery module furthered my knowledge that I gained from Khan Academy. Both the React and Redux modules certainly opened my eyes to just how much you can do with JavaScript. I learned that Redux is really helpful and essentially makes code reusable. Overall, this module certainly drove home the idea of combining JavaScript libraries to help achieve whatever you're trying to do in any given scenario. In the future, whenever I might code with JavaScript, this idea will certainly be at the forefront of my brain, as I've been able to see how useful it can be to add in different libraries. Even backing it up from just computer science, the idea of using outside sources to make your own work even better is something that is quite important to improve any work that you do.




AP Computer Science Principles




While I wasn't able to fully complete the entirety of the AP Computer Science Principles course from Khan Academy (mostly because they changed the entirety of the course right as I was close to completing it), I gained a lot of knowledge about computers, the internet, algorithms, programming, data and so much more. The course provided a lot of background on how exactly I am able to do anything with computers and made me appreciate the hardware and software that I am so fortunate enough to have access to. While there was a LOT of information that it became overwhelming at times in this course, it really was helpful in increasing my understanding and appreciation of computer science and technology, which is something that will stay with me in the future.
Note: There was no project for this module.
